# 📖 DailyWord | Inglés en 10 min/día

App estática para aprender vocabulario inglés de forma diaria. Sin backend, sin suscripciones, sin distracciones.

## ✨ Características
- 🎯 1 palabra/día con fonética, definición y ejemplos reales
- 🔊 Audio nativo con Web Speech API
- 🧠 Quiz interactivo de 3 opciones
- 🔥 Racha diaria guardada en `localStorage`
- 📱 100% responsive y ligera (<50KB)

npx serve .
